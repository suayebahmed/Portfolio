- Modern and Minimal UI Design
- Home, Projects, About and blog sections
- Fully Responsive
- Performances and SEO optimizations

# Dependencies

- svelte-icons
- @sveltejs/adapter-netlify

# Steps

```
$ git clone https://github.com/suayebahmed/Portfolio.git

$ cd SvelteKit-Portfolio

$ npm install

$ npm run dev
```
